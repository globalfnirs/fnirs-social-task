# fnirs-toolkit

## BRIGHT social task fNIRS analysis

Install MATLAB requirements:  
*Download the [Homer2](https://www.nitrc.org/projects/homer2) toolbox.*

Install Python requirements:  
`pip install matplotlib numpy pandas scipy seaborn statsmodels`

Scripts:
- `src/signal/JB_soc_preprocessing.m` social task fNIRS preprocessing
- `src/social/JB_soc_60mo.py` detailed social task fNIRS analysis at the 3-5 years age point
- `src/social/JB_soc_cross_sectional.py` social task fNIRS analysis at all the age points from 5 months to 3-5 years old
- `src/social/JB_soc_trajectories.py` study of trajectories with fNIRS on the social task from 5 months to 3-5 years old
